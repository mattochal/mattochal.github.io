MELR: Meta-Learning via Modeling Episode-Level Relationships for Few-Shot Learning
Nanyi Fei, Zhiwu Lu, Tao Xiang, Songfang Huang

This work proposes Modeling Episode-Level Relationship (MELR) which trains models by sampling two tasks per learning iteration, instead of one like in standard episodic meta-training. The query and support sets for the two tasks share the same classes but contain different instances.
